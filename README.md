# Professional Portfolio — GenAI PM Showcase

**Built with React, TypeScript, Tailwind CSS, and Vite**

A modern, responsive portfolio showcasing three AI-driven product projects: CareNote AI, LearnPath AI, and FinSense AI. Designed and engineered for performance, accessibility, and visual excellence.

## 🚀 Live Projects

Each portfolio project links to dedicated GitHub repositories:

1. **CareNote AI** — Clinical Documentation Copilot
   - Healthcare/MedTech | Jan–Apr 2025
   - [github.com/tekigowtham2204/CareNote-AI](https://github.com/tekigowtham2204/CareNote-AI)

2. **LearnPath AI** — Adaptive Skill-Gap Engine
   - EdTech/Learning | Aug–Dec 2024
   - [github.com/tekigowtham2204/LearnPath-AI](https://github.com/tekigowtham2204/LearnPath-AI)

3. **FinSense AI** — Conversational Financial Clarity Engine
   - FinTech | May–Aug 2024
   - [github.com/tekigowtham2204/FinSense-AI](https://github.com/tekigowtham2204/FinSense-AI)

## 📋 Features

✨ **Responsive Design** — Mobile-first, works across all devices  
⚡ **Performance** — Optimized build with Vite, lazy-loaded components  
🎨 **Modern UI** — Tailwind CSS with smooth animations via Framer Motion  
♿ **Accessible** — WCAG-compliant components from Radix UI  
🔗 **Smooth Scrolling** — Lenis scroll library for polished UX  
📱 **Component Library** — Pre-built UI components (buttons, cards, accordions, etc.)

## 🛠️ Tech Stack

| Layer | Technology |
|-------|------------|
| Build | Vite 6.3.5 |
| Runtime | React 18.3 + TypeScript |
| Styling | Tailwind CSS 4.1, Framer Motion |
| Components | Radix UI primitives |
| Icons | Lucide React |
| Routing | React Router 7 |
| Forms | React Hook Form |
| Animations | Motion / Framer Motion |
| Scroll | Lenis |

## 📥 Installation

```bash
git clone https://github.com/tekigowtham2204/Professional-Portfolio.git
cd Professional-Portfolio
npm install
```

## 🚀 Development Server

```bash
npm run dev
```

The app will run at `http://localhost:5173`

## 🏗️ Production Build

```bash
npm run build
```

Optimized build output in the `dist/` directory.
